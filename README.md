# CodeAlpha Hangman Game in Python

A simple **Hangman game** implemented in Python. Guess the word letter by letter before you run out of attempts! Includes fun emojis to make it more interactive. 

---

## Features
- Randomly chooses a word from a predefined list.
- Keeps track of guessed letters.
- Limits incorrect guesses to 6 attempts.
- Provides feedback for correct and incorrect guesses.
- Displays the current state of the word with underscores.

---
